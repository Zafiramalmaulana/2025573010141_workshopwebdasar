langkah membuat halaman web

  1. buat file index.html
  2. tulis struktur HTML
  3. tambahkan CSS untuk styling

Glosarium

HTML
hyper text markup language - bahasa markup untuk membuat halaman web
CSS
cascading style sheets - bahasa styling untuk mengatur tampilan halaman web
DOM
document object model - representasi HTML sebagai struktur pohon yang bisa dimanipulasi javascript
